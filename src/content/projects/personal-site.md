## Overview

This site is built with [Astro](https://astro.build), a static site generator that ships near-zero JavaScript by default.
Content is managed as Markdown files in the repository — no database or CMS required.

## Technical notes

- Static output deployed via GitHub Actions to GitHub Pages
- Content collections for type-safe frontmatter
- Plain CSS with custom properties — no utility frameworks
- Zero JS shipped for all non-interactive pages
